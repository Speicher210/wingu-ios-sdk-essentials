  • updatePhysical()

    Some of the channels did change physical characteristics
    If channels has some custom sorting or filtering this should be invoked in this place.
    func updatePhysicalInfo(:) will provide more details of objects that changed.

    Note This is a direct callback from CoreLocation method for beacons ranging. It may be called very often depends on your settings.

    Example: If currently in range is one region this method will get called every second (the same as CoreLocation native). If there is two ranged regions this callback will get called twice in second.
