WinguConfiguration

public struct WinguConfiguration

Sets of configuration that can be used to customize behaviour of wingu SDK.

  • returnOnlyChanelsWithContent

    By default winguSDK returns every channels found matching wingu criteria.
    This behavour can be changed using this flag and your delegate method will get only channels with valid content attached.
