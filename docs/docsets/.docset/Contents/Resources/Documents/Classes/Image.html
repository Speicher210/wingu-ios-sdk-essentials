Image

public class Image : NSObject, Codable

Contains informations about image. For managing images wingu is using Cloudinary. In this case uID will contain id of a picture in cloudinary.

  • type

    Image type.

    Declaration

    Swift

    public var type: Image.ImageType?
  • uID

    Unique identifier.

    Note: Every other unique identifier in wingu is in UUID format. This one is matched from 3rd-party service called Cloudinary.

    It can be used for direct call in cloudinary API.

    Swift

    public var uID: String?

  • url(for:)

    Create URL from width of the picture. Measured in points (not pixels)

    Note: This method take into consideration PPI of device later.

    Declaration

    Swift

    public func url(for width: CGFloat) -> URL?

    Parameters

    width

    Size of the expected image frame (ex. If image needs to be placed on iPhone 6 in UIImageView with size 100x100 width paramter should be 100)

    Return Value

    URL to selected image. If resource is invalid it will return nil

  • download(_:_:)

    Download image from URL passed with expected with, Should be match with method url(for:) to get right URL for size of the UIImageView

    Declaration

    Swift

    public func download(_ url: URL?, _ completion: @escaping(_ image: UIImage?) -> Void)

    Parameters

    url

    URL to download image. It’s recommended to use url(for:) function to return URL.
    completion

    Return UIImage object in occurance of success. Can be nil due to error for both client and server side.
