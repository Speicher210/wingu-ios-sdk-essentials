  • BeaconScanner

    Layer of abstraction for CoreLocation ranging & monitoring iBeacons. This is build using Singleton principle. There will never be a need to use more than one instance of BeaconScanner to work with wingu beacons. By default regions to monitor and ranging are equal and default wingu regions apply if not specyfied otherwise.
