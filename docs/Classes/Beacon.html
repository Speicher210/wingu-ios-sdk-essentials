Beacon

public final class Beacon : Channel

Representation of CoreLocation class CLBeacon with extra arguments from wingu portal.
This class combines fetched arguments from wingu portal such as Deck, Pack, Content and match it with physical characteristic of the beacon. All parameters from CLBeacon will be matched with the same naming to this class.

Note: This is not extend or override any CLBeacon parameters. Any changes in object of this class will not take any effect in physical beacon!

This class is marked as final, this shouldn’t need subclassed. If there is a need to change minor/major/uuid of physical beacon this should be done with beacons manufacturer dedicated application or SDK.
