NotificationsManager

public class NotificationsManager : NSObject

Allows you to receive UILocalNotification through wingu platform.

Note: wingu not sending push notifications only UILocalNotification whenever near Channel. There is no need to configure Apple Developer Certificate to handle Push Notification when app does not support it.

In order to receive notifications there might be different action required for different Channel type.

  • requestNotificationsPermission()

    Request notification permission for the end-user. It’s required to add this method to your project in order to receive Push Notification.
    Depens on end-user operating system notification can be rich or not. Under the hood, this method uses #available modifier so, there is don’t need to call separate version of this method even if there is need to support iOS 9.0. If more configuration required learn more about NotificationsManager.notificationsPermission(...)
