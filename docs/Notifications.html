Notifications

  • NotificationsManager

    Allows you to receive Notifications through wingu platform.

    Note: wingu not sending push notifications only whenever near Channel. There is no need to configure Apple Developer Certificate to handle Push Notification when app does not support it.
    wingu SDK is using UserNotifications framework

    In order to receive notifications there might be different action required for different Channel type.

  • PredefinedActions

    Actions for wingu notifications.

    Note: It’s wrapper for UNNotificationAction and it’s designed to work with wingu notifications. If custom notifications are in use there is most likely no need for using this class

    Object from this class should be used only with init method and it’s likely to use with WinguConfiguration struct.
