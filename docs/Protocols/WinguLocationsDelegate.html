  • updatePhysical()

    Some of the channels did change physical characteristics
    If channels has some custom sorting or filtering this should be invoked in this place.
    func updatePhysicalInfo(:) will provide more details of objects that changed.

    Note This is a direct callback from CoreLocation method for beacons ranging. It may be called very often depends on your settings.

    Example: If currently in range is one region this method will get called every second (the same as CoreLocation native). If there is two ranged regions this callback will get called twice in second.

  • winguUnseenChannels(_:)

    Unseen collection of channels. If the end user never display content this will return collection of unseen, available regions in the area.

    Note: There is no point of using this method if only Essential part of SDK is used. This only works with winguSDKContent.

    Viewing content is driven by some rules:

    • By view content action is understood by displaying wingu deck from winguSDKContent SDK
    • Whenever content updates in portal channel is treated as new one. It’s based on Content.lastSeen variable.
    • This is only valid for current locale. If language in portal was change for other language, this will not be returned here
    • Channel will be returned once per discovery, if there is a need to check physical information further use updatePhysicalInfo(_:) method
